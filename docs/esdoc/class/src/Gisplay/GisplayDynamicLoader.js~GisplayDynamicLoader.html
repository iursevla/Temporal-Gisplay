    private _isLibraryLoaded(libraryName: string): boolean source

    Verifies if the library CSS and JS are loaded into the HTML. If they are then there's no need to load it again so this will return true, otherwise, returns false.

    Params:

    NameTypeAttributeDescription
    libraryName string

    The name of the library (e.g. Mapbox).

    Return:

    boolean

    true if the js and css (if any) of the library were already loaded by the user, false, otherwise.

    See:

    • https://stackoverflow.com/a/37820644

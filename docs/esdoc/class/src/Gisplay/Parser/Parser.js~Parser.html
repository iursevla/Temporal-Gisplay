    public hasErrors(row: Array<Object>) source

    Verify if the row has errors or not. This method is called by the respective Parser for each row or just for one row. The programmer may decide to never call this method if he is sure the given options are all correct. Basically will look if all given options (visual variables, Geometry, time and extra variables) are according to it's specification in the Parsing options.

    Params:

    NameTypeAttributeDescription
    row Array<Object>

    Any row given to look for existing errors.

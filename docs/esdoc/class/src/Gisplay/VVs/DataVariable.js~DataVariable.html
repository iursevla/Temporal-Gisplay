    DataVariable

    Direct Subclass:

    MapVariable, TimeVariable

    Indirect Subclass:

    CategoricalVariable, ContinuousVariable

    Represents a variable. This variable has an external and an internal Name. External name is the name of the variable on the dataset. Internal name is the name the programmer gave to this variable.

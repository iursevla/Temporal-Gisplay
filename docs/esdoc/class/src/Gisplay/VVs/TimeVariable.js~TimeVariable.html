    public sortedTGsToIndexMap: Map<string|number, number> source

    The map with the string temporal granules and their respective index inm the sorted temporal granules array. E.g. Map (3) {"1 January 2016" => 0, "2 January 2016" => 1, "3 January 2016" => 2, "4 January 2016" => 3 }. Used to quickly search for the index of the start and end point of the time control. A value here is an index in the sortedTemporalGranulesArr.

    public sortedTemporalGranulesArr: Array<number|string> source

    The temporal granules sorted by Date. If its a number (year or day or value etc) then sorted by number.

    public strToTemporalGranuleMap: Map<string|number, string|number> source

    The map that contains the string that represents the date and the respective date (e.g. 2016_1_1 => 1 January 2016 OR 2016 => 2016) If the date is a number (e.g. year, monthOfYear, value etc) then the key and the value are equal (2016 => 2016). A value here will be a key in the temporalGranulesMap.
